# Gaming TodoList App

_Gaming TodoList App using React & Redux._

## App setup

```
npm install
```

## Runs the app

```
npm start
```

## Built With

* [React](https://reactjs.org/)
* [Redux](https://redux.js.org/)
* [React Redux](https://react-redux.js.org/)
* [Bootstrap](https://getbootstrap.com/)
* [Font Awesome](https://fontawesome.com/)
